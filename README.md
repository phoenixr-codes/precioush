# `preciou.sh`

`preciou.sh` (pronounced /ˈprɛʃəs/ "precious") is the Swiss Army Knife for POSIX shell programming.

> [!WARNING]
> This project is still very early in development.

## Features

- [x] `ansi`
- [ ] `asserts`
- [ ] `binary`
- [ ] `cli`
- [ ] `csv`
- [x] `core`
- [x] `development`
- [ ] `fs`
- [ ] `glob`
- [ ] `html`
- [ ] `http`
- [ ] `ini`
- [ ] `json` - JSON, JSON5, JSONC parser and writer
- [x] `log`
- [ ] `math`
- [ ] `prompt`
- [ ] `rand`
- [ ] `regex`
- [x] `syntax`
- [ ] `term`
- [x] `tests`
- [ ] `toml`
- [ ] `ui`
